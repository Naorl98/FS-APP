How to Use the Formula Sheet Creator

To create a formula sheet, upload one or more PDF summaries. The system will automatically merge multiple files into a single document.

For best results, it is recommended to upload summaries that contain mainly text and minimal images.

The “Resize Percentage” determines how much the pages will be scaled down. Lower values result in smaller text, but allow more content per page.

You can enter a custom name for the output file. If left blank, the system will generate a name based on the uploaded documents.

You may enable the “Leave the document as-is” option to skip the formula-sheet formatting algorithm. This will only resize and combine the pages without modification.

A highlight color can be applied to mark section titles, assuming the document uses bold or underlined formatting for headings.

Recommended settings: use a resize percentage between 5–15% and ensure your PDF contains mostly text-based content.

איך להשתמש ביוצר דף נוסחאות

ליצירת דף נוסחאות, יש להעלות קובץ PDF אחד או יותר. המערכת תאחד קבצים מרובים למסמך אחד באופן אוטומטי.

לתוצאה מיטבית, מומלץ להעלות סיכומים המכילים בעיקר טקסט ומעט תמונות ככל האפשר.

שדה “אחוז ההקטנה” קובע את גודל ההקטנה של הדפים. ככל שהאחוז נמוך יותר — הטקסט יהיה קטן יותר וייכנס יותר תוכן בכל עמוד.

ניתן להזין שם קובץ מותאם אישית. אם תשאיר את השדה ריק — יווצר שם אוטומטי לפי הקבצים שהעלית.

ניתן לבחור באופציה “השאר את המסמך כפי שהוא” כדי לבטל את עיבוד דף הנוסחאות — פעולה זו תבצע רק הקטנה וסידור של הדפים.

ניתן גם להפעיל צבע מרקר לזיהוי כותרות, במיוחד אם הן מודגשות או עם קו תחתון במסמך.

המלצה: השתמשו בהקטנה של 5–15% והעלו קובץ PDF טקסטואלי בלבד.
